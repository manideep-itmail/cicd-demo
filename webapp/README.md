# DevOps Portfolio Website

A professional multi-page static portfolio for a DevOps engineer. It is ready for GitHub Pages and does not require a build step.

## Customize

Edit `index.html` and replace:

- `Mani` with your preferred display name.
- `your.email@example.com` with your email address.
- LinkedIn and GitHub URLs with your real profile links.
- Project cards with your real case studies, metrics, and tools.
- Experience section with your actual roles, companies, and dates.
- Navigation links if you add or remove pages.

## Deploy on GitHub Pages

1. Create a new GitHub repository.
2. Upload the files from this folder to the repository root.
3. Commit and push.
4. In GitHub, open `Settings` > `Pages`.
5. Set the source to `Deploy from a branch`.
6. Choose the `main` branch and `/root`.
7. Save. GitHub will publish the site after the first Pages build completes.

## Local preview

Open `index.html` directly in your browser.
